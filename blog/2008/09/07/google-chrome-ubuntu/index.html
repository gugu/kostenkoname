Google Chrome в Ubuntu

Sep 7th, 2008

Версия Хрома под Linux ещё не готова, но потестировать хром можно и в Wine.

Нужна последняя весию Wine (на данный момент это 1.1.4). О том, как поставить последнюю версию для Вашего дистрибутива читайте на http://winehq.com. Чтобы проверить текущую версию wine нужно выполнить ЭТО: wine --version

Даунлоадер гугла не работает, нужно скачать ручками Google Chrome отсюда: http://gpdl.google.com/chrome/install/149.27/chrome_installer.exe

Заходим в директорию с инсталлятором и запускаем его под wine wine chrome_installer.exe

Инсталлятор Хрома очень быстро отработает и предложит запустить Хром. Если вы не хотите травмировать свой мозг - не запускайте его, мы ещё не завершили установку.

После этого нам нужно скачать winetricks для установки шрифтов: wget http://www.kegel.com/wine/winetricks chmod +x winetricks ./winetricks corefonts tahoma

Чтбы хром заработал, запускаем его с параметрами new-http и in-process-plugins: wine ~/.wine/drive_c/windows/profiles/$USER/Local\ Settings/Application\ Data/Google/Chrome/Application/chrome.exe --new-http --in-process-plugins

Для этого аццкого скрипта можно повесить alias и запускать по нему.

Во время работы встретил только один большой баг - HTTPS не работает :-(
